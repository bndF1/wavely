Features

Everything you need
in a podcast player.

Built with Google Podcasts as the reference experience, Wavely brings a clean, focused listening interface to every platform.

🔊

Beautiful Audio Player

Full-screen player with artwork, scrubber, playback speed (0.5×–2×), skip 30s, and a persistent mini-player docked above the tab bar.

NgRx SignalStore
📱

Truly Cross-Platform

One codebase. iOS native via Xcode, Android via Gradle, and a PWA for the web — all powered by Capacitor 8 with full hardware API access.

Capacitor 8
🌙

Dark Mode & Themes

System-aware dark mode with manual override. Persistent preference via localStorage. Smooth transitions using Ionic CSS custom properties.

Signal-based
📡

Offline & PWA Ready

Angular service worker caches the app shell, artwork (7 days), and iTunes API responses (1 hour). Works without a network after first load.

@angular/service-worker
🔍

Search & Browse

Real-time search with debounce against the iTunes Search API. Browse by category, trending shows, and new releases.

iTunes API
📚

Podcast & Episode Detail

Full podcast pages with episode list, subscribe toggle, and artwork. Episode detail with full description, scrubber, and speed controls.

Lazy-loaded routes

Built With

Latest stable
stack, always.

No legacy code. Every dependency pinned to the latest stable version, managed in a Bun-powered Nx workspace with full TypeScript strict mode.

Architecture

Clean, layered,
zero legacy.

Feature-first folder structure. Every component is standalone. Services are injected via inject(). State lives in NgRx SignalStores. No NgModules.

src/app/ ├── core/ ← services, guards, interceptors │ ├── services/ │ └── models/ ├── features/ ← lazy-loaded pages │ ├── home/ │ ├── browse/ │ ├── search/ │ ├── library/ │ ├── podcast-detail/ │ └── episode-detail/ ├── store/ ← NgRx SignalStores │ ├── player/ │ └── podcasts/ └── shared/ ← reusable components
